Retortawin - Site de Jogo de Azar estilo Tigrinho
Descrição

Retortawin é um site de apostas inspirado no estilo do famoso jogo "Tigrinho". A plataforma oferece uma interface intuitiva e moderna para os usuários aprenderem sobre o jogo, conhecerem as regras, verem o ranking e começarem a jogar rapidamente.

O projeto foi desenvolvido usando React com Vite para o ambiente de desenvolvimento, proporcionando uma experiência rápida e responsiva.

Funcionalidades

Cabeçalho estilizado com nome do site, navegação entre seções e botão “Jogar Agora”

Seções para descrição do jogo, regras e ranking dos jogadores

Imagem temática centralizada para criar a atmosfera do jogo

Botão interativo com efeitos visuais ao passar o mouse

Navegação simples com links internos que facilitam o acesso às seções

Tecnologias Utilizadas

React

Vite

JavaScript (ES6+)

CSS inline para estilos rápidos e personalizados
